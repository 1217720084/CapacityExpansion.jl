    TX1G

    Texas as one node, no existing capacity

    Data from Merrick et al. 2016

    Implemented with PV-price 0.5 /W fix: 2.388E+3 /MW cap: 5.16E+5 /MW

    Alternatively for price of 1.0/W edit .csv files and replace costs with fix: 4.776E+3 /MW cap: 1.032E+6 /MW

    Assuptions for transformation: demand mulitiplied with 1.48 solar devided by 1000
